Sobre Mim

Sou desenvolvedor Full Stack com foco em aplicações web modernas e interfaces performáticas. Trabalho com tecnologias front-end como React, TypeScript, JavaScript, HTML5, CSS3, além de bibliotecas como Axios, jQuery e frameworks como Tailwind CSS.

No back-end, atuo com Delphi utilizando o framework Horse para construção de APIs estruturadas e performáticas, aplicando boas práticas como arquitetura em camadas, separação de responsabilidades e padrões de projeto como MVC. Tenho experiência também com Java e Spring Boot, desenvolvendo sistemas escaláveis.

Tenho sólida experiência com os principais bancos de dados utilizados no mercado, incluindo PostgreSQL, SQL Server, Oracle e Firebird, e sei aplicar técnicas de otimização de queries, normalização e modelagem lógica/dimensional.

Sempre busco escrever código limpo, escalável e de fácil manutenção, adotando princípios como KISS, SOLID e Clean Code. Também me preocupo com a experiência do usuário final, garantindo interfaces responsivas e funcionais com tecnologias como Next.js, Bootstrap e react-hook-form (com validações via Zod).

No tempo livre, gosto de aprender coisas novas, praticar atividade física, tocar algum instrumento e explorar novas tecnologias — especialmente aquelas que agregam performance e produtividade ao dia a dia de um desenvolvedor.

Objetivos

Busco oportunidades para crescer como desenvolvedor e colocar em prática tudo que venho aprendendo. Estou aberto a novos desafios, especialmente projetos que envolvam:

  • Front-end moderno com React ou Next.js
  • Back-end estruturado com Delphi (Horse), Node ou Java (Spring Boot)
  • Desenvolvimento de APIs RESTful
  • Aplicação de princípios de arquitetura e clean code
  • Integração entre front-end e back-end com Axios ou Express, consumo de serviços e tratamento de erros

Meu objetivo é continuar evoluindo na área, contribuindo com soluções criativas e eficientes, sempre buscando qualidade, performance e crescimento contínuo!
